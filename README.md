# Course Project

The following gives and overview of a report structure. You can adjust to fit your needs.

- Abstract (0.5 page)
- Introduction (1 page)
- The Problem (1 page )
- Related work (2 pages)
- Your Idea or solution (1 page)
- The Details ( 5 to 8  pages)
- Conclusion and further work (1 page)
